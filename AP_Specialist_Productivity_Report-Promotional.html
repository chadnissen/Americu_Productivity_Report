The Challenge

AP Departments Are Flying Blind.

Invoices move through approval queues, but nobody sees the full picture until vendors escalate or auditors ask questions. The cost is invisible — until it isn't.

Without the Report
  • No visibility into where invoices are stuck — Site Manager or AP Approval — without digging through the workflow system manually
  • Managers resort to manual spreadsheet reconciliation to track specialist output and invoice counts
  • Zero SLA tracking: no way to know which site managers hold invoices for weeks or months
  • No aging analysis — dollar exposure buried in open queues goes unquantified until auditors ask
  • Workload imbalances across processors invisible until performance reviews, not in real time
  • Vendor escalations require manual queue-by-queue investigation with no centralized trail
With AP Specialist Productivity Report
  • Instant dashboard: every invoice, every approver, every day count — in a single browser view
  • Click any specialist or manager row to drill down instantly — no spreadsheets, no exports required
  • SLA color coding (green / yellow / red) flags overdue invoices the moment you open the page
  • Aging Summary buckets (0–3, 4–7, 8–14, 15–30, 31+ days) show dollar exposure in real time
  • Workload comparison across all processors in a single scrollable panel
  • Find any invoice in seconds: filter by vendor, sort by days, see exactly where it is stuck
The Solution

Every Invoice. Every Approver. Real Time.

The AP Specialist Productivity Report is a single-page web dashboard that connects directly to your ECMT Workflow database. It delivers a KPI summary strip with five key metrics, three click-to-drill-down rollup tables (By AP Specialist, By Site Manager, Aging Summary), and a 15-column sortable detail table with SLA color-coded processing days.


Deploy as a standalone Windows EXE or publish to IIS. Zero external dependencies. Zero schema changes. Connect, map your workflow fields, and you're reporting in minutes.


AP managers, finance controllers, CFOs, and IT administrators all get what they need from a single URL — secured by Windows Authentication and optionally locked to specific Active Directory groups.

How It Works

Six Steps. Deployed in Minutes.

From dropping the EXE to your first report run, the entire setup is browser-driven with no developer involvement required.

Step 01
Deploy
Drop the single self-contained EXE on any Windows machine or publish to IIS. No runtime install needed. ~101 MB, self-contained .NET 8.
Step 02
Connect
Enter your Workflow database credentials in the browser-based settings page. Test the connection with one click. Passwords are encrypted at rest with DPAPI.
Step 03
Map Fields
Select your workflow and map invoice fields (Vendor Name, Invoice Number, Amount, Date). Auto-detection handles most mappings automatically.
Step 04
Assign Queue Roles
Tag each workflow queue as Site Manager (First Approval) or AP/Senior (Second Approval). The report uses these to determine who approved and how long each stage took.
Step 05
Secure
Windows Authentication is built in. Optionally restrict access to specific Active Directory groups through the settings UI. No passwords to manage.
Step 06
Report
Click Run Report. Review KPIs and rollup summaries, click any row to drill down to invoice detail, sort any column, and export to CSV with one click.
Key Features

Built for AP Managers. Designed for Decisions.

Every capability was designed around a real AP workflow question. Not a data dump — an actionable management tool.

KPI Summary Strip
Five key metrics at a glance — Total Invoices, Total Dollar Value, Unique Processors, Unique Vendors, and Average Processing Days. Immediate situational awareness that updates dynamically when you drill down from rollup tables.
Rollup Tables with Click-to-Drill-Down
Three side-by-side summary panels — By AP Specialist (invoice count, avg days, open items), By Site Manager (assigned, avg days, pending, SLA%), and Aging Summary (age buckets with count and dollar value). Click any row to instantly filter the detail table below. A filter banner shows the active drill-down with a clear button.
Flexible Date Filtering & Presets
Presets (Month to Date, Quarter to Date, Year to Date, Last Month, Last Quarter) plus fully custom date ranges. Four date basis options: System Created Date, Invoice Date, Site Manager Approval Date, and AP Approval Date. Filter by individual processor and by status (All, Completed, In Progress).
15-Column Detail Table with SLA Color Coding
Every invoice listed with AP Processor, Vendor Name, Invoice Date, Invoice #, Amount, System Entry, Site Mgr Approver, Site Mgr Date, Mgr Days, AP Approver, AP Approval Date, AP Days, Total Days, Comments, and Status. Color-coded SLA indicators — green (0–3 days), yellow (4–30 days), red (31+ days) — make bottlenecks impossible to miss. Sortable columns, sticky headers, 25 per page.
Aging Summary with Dollar Exposure
In-progress invoices bucketed by age — 0–3, 4–7, 8–14, 15–30, and 31+ days. Each bucket shows count, dollar value, and how many invoices are waiting at the Site Manager stage. Color-coded age bands highlight where money is stuck. CFOs can instantly see financial exposure by aging tier.
One-Click CSV Export
Export the current filtered and sorted view to CSV with UTF-8 BOM encoding — ready for Excel, compliance audits, or downstream analytics. Exports only the visible filtered rows, so managers can drill into a specific subset and export just those invoices.
Dynamic Workflow Mapping
Map any ECMT workflow to the report through the browser UI. Auto-detection matches common field names (Vendor Name, Invoice Number, Amount, etc.). Assign queue roles — Site Manager (First Approval) or AP/Senior (Second Approval) — to track multi-level approval stages. Add or remove workflow mappings without code changes. Supports multiple workflows per deployment.

Configuration Details
Database Connections Workflow DB (required) and AppEnhancer DB (optional) configured through the browser UI. One-click Test Connection validates credentials before saving. All passwords encrypted at rest using Windows DPAPI (machine-bound).
Workflow Field Mapping UI-driven mapping of invoice fields to workflow table columns. Auto-detection matches common field names on first load. Map Vendor Name, Invoice Number, Invoice Amount, Invoice Date (required), plus DocID, Workflow Status, and Workflow Queue (optional).
Queue Role Assignment Tag each workflow queue as First Approval (Site Mgr), Second Approval (AP/Senior), or Not Used. The report uses these role assignments to attribute approval steps and calculate stage-level day counts.
Access Control Restrict access to specific Active Directory groups using a search-and-add interface. Leave empty to allow all authenticated Windows domain users.
Security & Compliance

Enterprise Security. Zero Extra Infrastructure.

Windows Authentication, DPAPI credential encryption, and AD group access control — built in, not bolted on.

Windows Authentication
Negotiate/Kerberos authentication via IIS or self-hosted. No login form, no separate password management. Users are silently authenticated via their domain credentials — seamless SSO for all domain users.
DPAPI Credential Encryption
Database credentials are encrypted at rest using Windows DPAPI, machine-bound. Credentials cannot be decrypted on another host. No plaintext passwords in configuration files or the registry.
AD Group Access Control
Restrict report access to specific Active Directory groups via the settings UI. Leave empty for open access to all authenticated domain users. Add or remove groups at any time without redeploying.
