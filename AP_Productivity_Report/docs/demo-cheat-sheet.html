Internal Use

Opening (30 seconds)

“This is a self-service reporting tool that gives AP managers real-time visibility into invoice processing performance — who’s doing the work, how fast approvals are moving, and where invoices are stuck.”

Demo Flow (5–8 minutes)

  1. Run the report — Select a date preset (YTD works well) and click Run Report. Point out the page loads in seconds with no install required.
  2. KPI cards — Walk across the 5 cards: Total Invoices, Dollar Value, Unique Processors, Unique Vendors, Avg Processing Days. “This is your at-a-glance executive summary.”
  3. Rollup tables — Show the three summary tables:
    • By AP Specialist — who has the most volume, who has invoices stuck
    • By Site Manager — approval speed and SLA compliance %
    • Aging Summary — click the 31+ days row (high-impact moment — instantly isolates overdue invoices)
  4. Click-to-filter — Click any rollup row. Show the blue filter banner, then show KPIs recalculate instantly. Click Clear Filter to reset.
  5. Status filter — Switch between All / In Progress / Completed. KPIs and rollup tables update without a page reload.
  6. Sort & browse — Click a column header (Amount or Total Days work well). Show pagination if many rows.
  7. Date basis — Switch from “System Created Date” to “Invoice Date” or “Approval Date.” “Lets you analyze the same data from different perspectives.”
  8. CSV export — Click Export. “Opens directly in Excel — great for month-end audit or ad-hoc analysis.”
Pro tip: If time is short, steps 1–4 are the money shots. Steps 5–8 are supporting detail.

Common Questions

Q: Does this require a separate database?
No — it reads directly from the existing ECMToolbox Workflow database. No data duplication.
Q: Can we customize which fields map to which columns?
Yes — the Settings page lets you map any workflow field to the report columns. Auto-detection handles most setups.
Q: Who can access it?
Controlled by Active Directory groups. You decide which AD groups can see the report.
Q: Can we change the SLA thresholds?
That’s on the roadmap — configurable per-workflow so each team sets their own targets.
Q: What about multiple approval steps?
Currently supports two-step (Site Mgr + Senior). Configurable N-step approvals are planned.
Closing line: “This gives your AP managers the visibility they need without waiting on IT to run a report. It’s self-service, runs on your existing data, and deploys in minutes.”
